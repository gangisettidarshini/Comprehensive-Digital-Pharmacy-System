# Pharmacy System (Java)

The Pharmacy System is a console-based Java application that helps manage basic pharmacy operations such as maintaining medicine records, updating stock, and generating simple actions for inventory control. The system is menu-driven and easy to use, making it ideal for beginners learning Java, Object-Oriented Programming (OOP), and data handling.

---

## ✨ Features

### ✔️ Add New Medicine
Users can enter:
- Medicine ID  
- Medicine Name  
- Quantity  
- Base Price  

The system creates a new medicine object and stores it in the inventory.

### ✔️ Update Medicine Quantity
Allows the user to:
- Enter a medicine ID  
- Enter a new quantity  
- Update existing stock in the inventory

### ✔️ Display Medicine List  
Shows all available medicine details stored in the inventory.

### ✔️ Remove Medicine  
Delete a medicine record based on its ID.
